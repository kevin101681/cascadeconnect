# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1p9T4-RIarBtSeiN1R9L9NgmihdsGh1i6

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Copy `env.example` to `.env.local` and fill in your environment variables (see `env.example` for required variables)
3. Run the app:
   `npm run dev`
